# Compatibility Matrix

This app follows semantic versioning and is designed to work with specific versions of Nautobot. Each release of the Nautobot IGP Models app is tested against the supported Nautobot versions listed below.

The app receives bug fixes and feature updates for the current major version. When a new major version of Nautobot is released, the app will be updated to support it, and support for older Nautobot versions may be deprecated according to Nautobot's support policy.

| Nautobot IGP Models Version | Nautobot First Support Version | Nautobot Last Support Version | Python Version Support |
| ------------- | -------------------- | ------------- | ------------- |
| 1.0.X         | 3.0.0                | 3.9999        | 3.10 - 3.13   |

## Upgrade Path

When upgrading between versions, always review the [Release Notes](release_notes/index.md) for breaking changes and migration instructions.
